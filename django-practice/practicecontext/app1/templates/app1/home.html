Built-in Django Template Filters

  1. Join Filter

    {{fvS|join:"-"}}

  2. Date Filter

    {{date| date:"D d M Y"}}

  3. Default Filter

    {{"" | default:"an empty string" }}

  4. Add Filter

    {{age | add:"5" }}

  5. Capfirst Filter

    {{'django' | capfirst}}

  6. Cut Filter

    {{"Python And Django is Fun" | cut:" "}}

  7. Dictsort Filter

    {{ dics | dictsort:"name"}}

    {{ dics | dictsort:"age"}}

  8. Escape Filter

    {{ escap | escape }}

  9. First Filter

    {{ fvS | first }}

  10. Last Filter

    {{ fvS | last}}

  11. Length Filter

    {{ fvS | length}}

  12. Linenumbers Filter

    {{ line | linenumbers }}

  13. Lower Filter

    {{ name | lower }}

  14. Upper Filter

    {{ name | upper }}

  15. Title Filter

    {{ clg | title }}

  16. Random Filter

    {{ fvS | random }}

  17. Slice Filter

    {{ slicees | slice:":50" }}

  18. Time Filter

    {{ date | time:"H:i" }}

  19. Timesince Filter

    {{ date | timesince }}

  20. Truncatechars Filter

    {{ slicees | truncatechars:50 }}

  21. Wordcount Filter

    {{ slicees | wordcount }}

  22. Truncatewords Filter

    {{ slicees | truncatewords:2 }}

  23. Striptags Filter

    {{ escap | striptags}}

  24. Pluralize Filter

    You have {{ num_tomato }} tomato {{ num_tomato | pluralize:"es" }}.

  25. Make_list Filter

    {{ name | make_list }}
